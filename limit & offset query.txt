⭐ LIMIT query on a table

👉 limit records fetching (for ex pagination)
command :
 select * from "Tablename" limit 4  (will show 4 records)
 
⭐ OFFSET query on a table

👉 OFFSET records fetching (for ex pagination ,leave n records from start)
command :
 select * from "Tablename" limit 4 offset 2  (will show 4 records leaving 2 from starting)
